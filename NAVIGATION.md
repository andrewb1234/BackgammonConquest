# Project Knowledge Index

## Purpose
Provide a routing map for locating rules and logic without scanning the full repository.

---

## Directory Map

- Need visuals / lore → `aesthetic_narrative.md`  
- Need startup / matchmaking flow → `entry_flow.md`  

- Need win conditions or turn loops → `game_flow.md`  
- Need core movement rules (Backgammon logic) → `backgammon_rules.md`  

- Need currency, costs, or escalation logic → `economy.md`  
- Need node / map modifiers → `planets.md`  

- Need loadout or item behavior → `items.md`  
- Need buff / debuff rules → `status_effects.md`  

- Need data structures → `state_model.md`  
- Need server authority / sync → `networking_sync.md`  
- Need WebSocket schemas → `websocket_payloads.md`  

- Need client UI routing → `ui_state_machine.md`
