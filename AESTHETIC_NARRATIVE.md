# UI Styling and Narrative Terminology

## Purpose
Define visual constraints and strict string mapping for standard Backgammon terminology.

---

## Visual Constraints

### Global Theme
- Industrial, grim-dark sci-fi aesthetic  
- "Lo-fi" interface design:
  - CRT scanlines  
  - Monochrome green/orange text  
  - Heavy metallic textures  

### Iron Hegemony Faction
- Dominant colors: rust-red, dark grey  
- Pieces: heavy gear-like discs  

### Solar Covenant Faction
- Dominant colors: gold, marble-white  
- Pieces: glowing blue protective auras  

### Game Board
- Rendered as a holographic tactical table  
- Triangles: stylized as distinct landing zones or trenches
